## 🎨 Starbucks 设计系统亮点

### 核心特点

1. **四级绿色品牌系统**
   - Starbucks Green: `#006241` (历史品牌绿)
   - Green Accent: `#00754A` (主要 CTA 颜色)
   - House Green: `#1E3932` (深色背景)
   - Green Uplift: `#2b5148` (次要装饰)

2. **暖中性画布**
   - Neutral Warm: `#f2f0eb` (温暖奶油色)
   - Ceramic: `#edebe9` (陶瓷色)
   - 参考真实咖啡馆材质

3. **专有字体系统**
   - SoDoSans: 主字体 (专有)
   - Lander Tall: Rewards 页面衬线字体
   - Kalam: Careers 页面手写字体
   - 字间距: `-0.16px` (紧密感)

4. **圆角设计**
   - 按钮: 50px 全圆角 pill
   - 卡片: 12px 圆角矩形
   - 浮动 CTA: 56px 圆形

5. **浮动 Frap CTA**
   - 特色设计元素
   - 多层阴影效果
   - `scale(0.95)` 按压交互

## 💡 实际应用建议

### 不要直接复制

❌ 错误做法:
```css
/* 直接复制 Starbucks 颜色 */
--primary: #006241;
--background: #f2f0eb;
```

✅ 正确做法:
```css
/* 学习四级颜色系统的思路 */
--primary: #000000;          /* 主要品牌色 */
--primary-accent: #1a1a1a;   /* 强调色 */
--primary-dark: #000000;     /* 深色变体 */
--primary-light: #f5f5f5;    /* 浅色变体 */
```

### 提取设计原则

从 Starbucks 学习：
1. **颜色分级** - 不只是"绿色"，而是四级绿色系统
2. **金色保留** - 高价值元素专用色
3. **暖中性** - 用色彩营造氛围
4. **上下文字体** - 不同场景使用不同字体
5. **圆角一致** - pill 和卡片的统一圆角体系

应用到当前 Figma 系统：
1. ✅ 已有颜色分级 (background, foreground, muted)
2. ✅ 可以考虑"专用强调色"概念
3. ⏳ 可选: 考虑暖色调变体
4. ✅ 已有字体层次系统
5. ✅ 已有统一圆角系统

### 混合使用技巧

结合 Figma + Starbucks 的优点：

**从 Figma 保留**:
- ✅ 极简配色 (黑白为主)
- ✅ 虚线焦点轮廓
- ✅ 变量字重系统
- ✅ 负字间距设计

**从 Starbucks 借鉴**:
- ⏳ 浮动 CTA 设计概念
- ⏳ 多层次阴影（选择性使用）
- ⏳ `scale(0.95)` 按压交互
- ⏳ 圆角卡片 (12px 而非 8px)
